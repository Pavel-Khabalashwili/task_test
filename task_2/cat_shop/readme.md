# Описание проекта

Этот проект представляет собой веб-приложение на Flask, которое отображает информацию о кошках. Приложение подключено к базе данных PostgreSQL и использует SQLAlchemy для работы с базой данных.

## Установка

1. Склонируйте репозиторий на локальную машину:`git clone`
2. Соберите и запустите контейнеры с помощью Docker Compose:`docker-compose up --build`
3. Откройте браузер и перейдите по адресу http://127.0.0.1:5000 для просмотра веб-приложения.

## Структура проекта

* app.py: основной файл приложения Flask.
* templates/index.html: шаблон главной страницы, отображающий список кошек.
* templates/cat_detail.html: шаблон страницы с подробной информацией о кошке.
* static/images/: директория с изображениями кошек.
* requirements.txt: файл с зависимостями Python.
* Dockerfile: файл для сборки Docker-образа.
* docker-compose.yml: файл для настройки и запуска контейнеров с помощью Docker Compose.
* init.sql: файл для инициализации базы данных PostgreSQL.
